## UPGRADE FOR DEBIAN & UBUNTU 
Masukkan perintah dibawah jika anda menggunakan OS Debian Version 10 & Ubuntu 20
```
apt update && apt upgrade -y --fix-missing && update-grub && sleep 2 && apt -y install xxd && apt install -y bzip2 && apt install -y wget && apt install -y curl && reboot
```


## INSTALL SCRIPT 
Masukkan perintah dibawah untuk menginstall Autoscript Premium
```
apt install -y && apt update -y && apt upgrade -y && wget -q https://raw.githubusercontent.com/Ridhan354/instalasi/main/setup.sh && chmod +x setup.sh && ./setup.sh
```
